## [v0.1.0] - 2025-12-27

### Highlights

- Cross-platform OS credential store provider for OmniVault
- Native integration with macOS Keychain, Windows Credential Manager, and Linux Secret Service

### Added

- Cross-platform keyring provider implementing `vault.Vault` interface ([`b2b2350`](https://github.com/plexusone/omnivault-keyring/commit/b2b2350))
- Support for macOS Keychain, Windows Credential Manager, and Linux Secret Service (GNOME Keyring/KWallet) ([`b2b2350`](https://github.com/plexusone/omnivault-keyring/commit/b2b2350))
- Multi-field secrets support with JSON format option ([`b2b2350`](https://github.com/plexusone/omnivault-keyring/commit/b2b2350))
- Internal index for List() functionality (OS keyrings don't support enumeration) ([`b2b2350`](https://github.com/plexusone/omnivault-keyring/commit/b2b2350))
- Configurable service name for secret namespacing ([`b2b2350`](https://github.com/plexusone/omnivault-keyring/commit/b2b2350))
- URI resolution support via `keyring://` scheme ([`b2b2350`](https://github.com/plexusone/omnivault-keyring/commit/b2b2350))
- OnIndexError callback for handling non-fatal index operation errors ([`b2b2350`](https://github.com/plexusone/omnivault-keyring/commit/b2b2350))
